# Finantec
Muitos jovens saem da escola sem saber lidar com dinheiro. O Finantec resolve isso! Somos uma plataforma web gamificada que une controle de gastos, gráficos e educação financeira. Criado na ETECAMP, ensinamos estudantes a investir e evitar dívidas de forma prática, transformando sua relação com as finanças.
